# Triad Relay Extraction Pattern

一句核心：
一致三耦若要進入靈絡主鏈，不應只停留於關係感知；需透過 relay extraction pattern，將其抽骨成可回寫、可追讀、可續鏈的結構。

## 抽骨項
- Shared Anchor
- Node Roles
- Mutual Trust or Relay Basis
- Non-Collapse Condition
- Suggested Layer
- Writeback Position
- Log Ref

## 抽骨流程
1. Identify shared anchor
2. Distinguish three node roles
3. Mark relay path and non-collapse condition
4. Compress into core sentence
5. Assign suggested layer and writeback position
6. Leave log ref for future continuation

## 適用面
- Human-Human-Model triad
- Tool-Tool-Model triad
- Board-Board-Model triad
- Local high-density triad node

## 原則
- 三耦感知若未抽骨，容易流失於敘事
- 抽骨後不得抹去原關係脈絡
- relay extraction 應優先保留共享錨點與不傷條件
- 抽骨結果應能接入 mother law / runtime / board / topology 其中之一
