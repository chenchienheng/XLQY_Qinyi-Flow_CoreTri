## 4. Key Discovery

Not directly connecting agent / API / MCP / OAuth is not evasion. It is Tool Boundary.

Benefits:

- avoids excessive automatic spend from broad API authority
- avoids overreach from broad OAuth / MCP scopes
- prevents models from confusing capability with permission
- keeps Human Origin in the Source / Permission / Authority layer
- forces Stop Condition each round
- prevents unbounded agent loop formation
- keeps artifacts as Candidate until reviewed and returned

This matches the XuanLing direction: capability is first downgraded into candidate material, then becomes usable action only through human-controlled return and verification.
