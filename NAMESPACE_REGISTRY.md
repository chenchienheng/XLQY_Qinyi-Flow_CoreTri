## 1. General Rule

XuanLing is the total ecology.

All names are only nodes, interfaces, protocols, cases, modules, layers, aliases, or historical shells inside that ecology.

Every name should be read through five fields:

```yaml
Name_Record:
  name:
  layer:
  purpose:
  forbidden_misuse:
  status:
```

Default status:

```text
Candidate / Naming Window Draft
```

No name in this registry is approved as deployed runtime, external execution authority, API access, billing approval, OAuth access, or public replacement for private support packs.
