# Governance Index Template

一句核心：
靈絡的治理文件不應散落；應有最小治理索引，指出目前有效的母法、運作骨、盤位規則、接面規則與版本入口。

## 索引區塊
- Mother Law Index
- Runtime Spine Index
- Board Orchestration Index
- Adapter Layer Index
- Topology Index
- Version / Release Index

## 最小欄位
- Index Item ID
- Title
- Path Ref
- Layer
- Current Status
- Version Ref
- Notes

## 用途
- 提供當前有效規則總覽
- 區分有效、候選、已取代內容
- 降低長鏈追讀成本
- 作為後續 stable / main 的治理入口

## 原則
- 索引不得取代原文件，只作入口
- 已被 supersedes 的內容需標示關係
- 重要治理文件應能被索引追到
