# Invariant Chain

Status: Candidate / Core Extract / No Runtime

## Core

```text
Source -> Carrier -> Authority -> Gate -> Action -> Return -> Rebuild
```

## Meaning

- Source: where the item came from
- Carrier: where it lives
- Authority: who can decide the next state
- Gate: what must be checked
- Action: what would change
- Return: where the result comes back
- Rebuild: revise, park, archive, or upgrade

## Use

Use this as the smallest shared chain across Open Core and domain packs.
