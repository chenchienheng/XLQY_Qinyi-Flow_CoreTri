# Qinyi LOR Signal Return Template v0.1

Status: Candidate / Return Template / No Runtime

## Core

Qinyi_LOR converts signals into human-readable context, pressure visibility, authority boundary, and return-ready cards.

## Template

```yaml
Qinyi_LOR_Signal_Return:
  date:
  source_signal:
  status: "Candidate / Qinyi Return / No Runtime"
  facts: []
  inferences: []
  to_verify: []
  human_context:
  pressure_risk:
  wording_adjustment:
  authority_boundary:
  candidate_actions: []
  manual_needed: []
  return_to:
  one_line_summary:
```

## Red Doors

- Signal summary is not decision.
- Human wording cannot hide authority changes.
- Candidate action is not approved action.
- Return is not closeout.
