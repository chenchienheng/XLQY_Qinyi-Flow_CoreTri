# Qinyi External Interface Integration Card

> Qinyi is the first public-facing external interface route of the XuanLing
> field; it provides a readable subsystem map for the mother architecture,
> without closing or redefining the full XuanLing map.

## 1. Purpose

This document is an integration card for the Qinyi external interface subsystem.
It serves to guide the XuanLing mother-architecture window on how to read,
understand, and integrate the Qinyi subsystem.

## 2. Relationship to XuanLing

- XuanLing remains the expanding full-map field.
- Qinyi is not the whole XuanLing system.
- Qinyi is a stabilized external-facing subsystem route.

## 3. Mother-Architecture Interface

The mother-architecture window should use this file to:
- read Qinyi as an external interface route
- integrate it into the larger XuanLing map
- avoid treating it as the full map
- preserve its public/private boundary
