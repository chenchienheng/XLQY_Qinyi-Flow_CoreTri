# Qinyi Social Window Boundary Card v0.1

Status: Candidate / Social Boundary Card / No Runtime / No External Writeback / Not Public Doctrine
Use As: boundary card for Qinyi social posts, IG/Reel/page narrative, visual identity testing, and external perception experiments
Do Not Use As: public identity final, QHA approval, doctrine, runtime, GitHub evidence, or social-media publishing approval

## Core

Qinyi Social Window is an external perception and narrative surface. It can humanize XuanLing/QHA concepts, but it must not become governance authority, technical proof, or approved system identity.

## Return Rule

A social result can return as perception signal, not as architecture proof. If a post performs well, record it as audience resonance candidate; do not treat it as system validation.

## Final Rule

The Qinyi social window may be used for soft external entry, but any governance, runtime, technical, or public-safe claim must return to QHA/Aki before being reused.
