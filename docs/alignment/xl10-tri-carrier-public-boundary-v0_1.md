## One-line Core

XL10 is the public generic coordination and return-schema home. Live task state, private source context, complete window returns, and native evidence remain in their authorized carriers and are referenced rather than mirrored here.

## Tri-Carrier Position

```text
Conversation / Work
  -> forms source, meaning, judgment, and bounded commission

Authorized working carrier
  -> holds live tasks, returns, evidence pointers, and protected context

XL10
  -> versions selected generic schemas, protocols, templates, and de-identified patterns
```

The private working carrier may be newer than this repository. A repository commit does not become current operational truth merely because it exists.

## Content Treatment Rules

```yaml
KEEP_IN_XL10:
  - generic role maps
  - generic task and return schemas
  - public-safe audit checklists
  - bounded build packet templates
  - reusable circulation protocols
  - de-identified cross-domain patterns with explicit boundaries

RETURN_TO_PROTECTED_CARRIER:
  - private relationship or family source
  - complete window returns containing local context
  - company originals or tenant information
  - credentials, personal data, customer data, or unredacted evidence

REVIEW_BEFORE_KEEPING_PUBLIC:
  - personal decision examples
  - music and social-window cases
  - learning returns derived from protected sources
  - M365 build cards or company-adjacent designs
```
