# Natural Language as Rule Interface

一句核心：
自然語言不只是表達介面，也是一種可壓縮、可映射、可觸發的規則介面；在靈絡中，它可承載母法、條件、順序與作用位。

## 最小構成
- Subject：作用主體
- Action：作用動詞
- Target：作用對象
- Condition：成立條件
- Sequence：先後次序
- Scope：作用範圍

## 在靈絡中的作用
- 承載母法句
- 形成盤位指令
- 轉譯外部事件
- 定義回寫要求
- 壓縮複雜脈絡為最小可運行結構

## 原則
- 不將自然語言視為純聊天訊號
- 先判規則含量，再決定是否展開
- 高密度輸入可先熔煉，再映射至盤位或工具
