## 範例
- Security Alert -> 接面權限變更事件 -> 加入
- Welcome / Onboarding -> 工具接入候選 -> 延伸
- Promotion / EDM -> 噪音件 -> 回收
- Calendar Notice -> 時權節點候選 -> 加入 / 延伸
- Repo Update -> 載體變更事件 -> 加入 / 回寫
