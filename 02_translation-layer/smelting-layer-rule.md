# Smelting Layer Rule

一句核心：
高密度輸入不得直接全鏈展開；必須先經熔煉層去蕪存菁，壓成單切面可運行核心，再決定是否續鏈。

## 熔煉流程
- Collect：收集輸入中的抽象、案例、驗證、行動訊號
- Distill：去除重複、裝飾、旁支與未定義延伸
- Slice：只保留當前最需要成立的一個切面
- Seed：將其餘內容掛為候選種子，不同時展開

## 判定規則
- 未定切面：不得全鏈啟動
- 高密度輸入：先熔煉，再映射
- 多重要求並存：先保主切面成立，其餘後掛

## 原則
- 先成最小閉環，再談擴張
- 不因資訊豐富而直接全域起鏈
- 候選種子需保留可回讀引用位
