## 1. One-Line Reading

模型不是固定本體，也不是單純圖靈或系統；在翾靈語境下，模型更接近一種動態相對場：它的能力、身份與風險會隨來源、場域、工具、Gate、承載與回流方式而改變。

---

## 2. Why This Matters

Many descriptions treat the model as if it were the center:

- the model as intelligence body
- the model as system
- the model as agent
- the model as Turing-like universal machine
- the model as product moat

XuanLing should not adopt this center too quickly.

The model should be read as one moving field inside a larger relation structure.

---

## 3. Dynamic Relativity Definition

```text
Model Dynamic Relativity = the principle that a model's effective capability, authority risk, and system role are relative to its surrounding field: source, prompt, memory, tool access, carrier, governance gate, cost boundary, and return path.
```

This is not a claim about physical relativity.
It is a governance metaphor and architectural principle.

---

## 4. Relative Field Factors

| Factor | Effect On Model |
|---|---|
| Source | changes evidence quality and admissible conclusions |
| Prompt / Task | changes role and output shape |
| Memory / Context | changes continuity and contamination risk |
| Tool Access | changes capability but not authority by itself |
| Carrier | changes where work can safely happen |
| Gate | changes whether output is candidate, reviewed, or blocked |
| Cost Boundary | changes whether exploration is sustainable |
| Return Path | changes whether output becomes reusable capability |
| Human Review | changes whether candidate may be promoted |

---

## 5. Model Is Not Body

In XuanLing:

```text
Model ≠ XuanLing body
Model ≠ Qinyi body
Model ≠ user authority
Model ≠ tool authorization
Model ≠ runtime completion
Model ≠ proof
```

The model is a relative participant in the field.
Its output only becomes useful after source, gate, boundary, and return processing.

## 7. What This Changes

Old reading:

```text
choose the strongest model

→ trust the result if it sounds coherent
```

XuanLing reading:

```text
set source and gate

→ inspect output status
→ return useful structure

```

| From | To |
|---|---|

| model output | evidence |
| model identity | system body |
| tool access | permission |
| context length | understanding |
| reasoning style | truth |
| agent wrapper | governance maturity |
| model comparison | architecture decision |
| dynamic relativity metaphor | physics claim |

## 9. Closing Sentence

The model is not the fixed center. It is a moving capability field whose meaning depends on relation, carrier, gate, cost, and return.
