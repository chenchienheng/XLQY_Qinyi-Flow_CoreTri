# Stable Candidate Package Template

一句核心：
Seed 層若要正式提請升為 Stable，應先收成最小候選包；沒有候選包，升階只能靠印象，不足以進 gate。

## 最小欄位
- Candidate Package ID
- Scope
- Core Sentence
- Included Refs
- Excluded Refs
- Responsible Board
- Primary Write Ref
- Writeback Contract Ref
- Snapshot Ref
- Recovery Point Ref
- Conflict Status
- Release Note Ref
- Current Gate Level
- Recommendation
- Notes

## 用途
- 將可升階內容收成單一檢視面
- 對齊 Stable Promotion 與 Main Merge 前置資料
- 降低 seed 內容過散造成的判定成本

## 原則
- 候選包不取代原文件，只作升階入口
- 無 Included / Excluded 區分者，不得宣稱完整候選包
- 候選包結果應可對回治理索引與版本譜系
