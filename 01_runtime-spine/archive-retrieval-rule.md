# Archive Retrieval Rule

一句核心：
封存不等於消失；被封存的窗口、規則、主案卡與日誌，應保留可追讀、可引用、可再啟用的取回路徑。

## 適用對象
- Archived Window
- Archived Project Card
- Archived Rule
- Archived Log or Snapshot

## 最小欄位
- Retrieval ID
- Archived Ref
- Requested By
- Retrieval Reason
- Time
- Restore Mode
- New Ref if Reactivated
- Log Ref

## Restore Mode
- Read Only
- Reopen as New Window
- Recover as Reference
- Split and Reattach

## 規則
- 封存內容不得直接改寫原歷史
- 再啟用應建立新引用或新窗口
- Read Only 只供追讀，不得回寫原主寫位
- 取回結果需能對回原封存來源
