## 1. One-Line Reading

「動態相對論」比「模型是圖靈機」或「模型是系統」更貼近翾靈正在描述的東西；但它不是物理相對論，而是脈絡、位置、時間、權限與回流之間的動態相對關係。

---

## 2. Why Not Only Turing Or System

Calling a model a Turing machine mainly captures computability.

Calling a model a system mainly captures structure.

XuanLing handles something broader:

- the same sentence means different things in different windows
- the same tool has different capability under different permissions
- the same document has different weight under Candidate, Approved, or Execution status
- the same person cannot be handled the same way in life language, governance language, and faith language
- the same model is not the same kind of entity in chat, Codex, agent, runtime, API, and document governance

Therefore the problem is not only computation or structure.
It is dynamic relation under source, field, authority, state, carrier, time, and return.

---

## 3. Not Arbitrary Relativism

Dynamic relativity does not mean anything can mean anything.

Relative does not mean arbitrary.

Invariant constraints remain:

| Relative Layer | Non-Relative Boundary |
|---|---|
| position is relative | invariant core is not relative |
| language is relative | boundary is not relative |
| tool role is relative | sovereignty is not relative |
| status may vary | Candidate is not Approved |
| field may change | people must not be flattened |
| carrier may vary | authorization must not be assumed |
| return form may vary | Return Packet is not Closeout |

This separates XuanLing dynamic relativity from ordinary relativism.

## 4. Bottom Formula

A first working formula:

```text
Meaning = Source × Field × Authority × State × Carrier × Time × Return
```

Chinese:

```text
意義 = 來源 × 場域 × 權限 × 狀態 × 承載 × 時態 × 回流
```

Interpretation:

```text
The same object changes meaning when source, field, authority, state, carrier, time, or return path changes.
```
