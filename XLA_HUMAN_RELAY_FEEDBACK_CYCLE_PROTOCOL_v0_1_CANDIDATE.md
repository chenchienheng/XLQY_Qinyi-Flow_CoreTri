### Codex Guardrail

Codex remains workspace support-cell feedback. It is not final authority, autonomous executor, or doctrine owner. Codex returns must go back to Qinyi Main for cleaning and Architecture Window for candidate positioning.

### Cross-Model Feedback Contract

This aligns with Six-Face logic:

```text
Source / Status / Permission / Authority / Carrier / Return
```

The protocol is the process version of that contract.

### Agent Mode Cost Gate
