# Notion Public Board Mirror Rule

一句核心：
若 Notion 作為 ⓪ 公盤首輪候選，其角色應限定為入口鏡像與整合視圖，不得反向搶占 GitHub 主寫位。

## 角色
- Mirror Entry View
- Status Aggregation View
- Handoff Visibility Layer

## 可承接內容
- Governance Core Entry
- Current Blockers Summary
- Tool Status Summary
- Next Action Queue
- Handoff Target Display

## 不承接內容
- Rule 主寫
- Version 主寫
- Freeze / Release 主寫
- 衝突解決最終主寫

## 原則
- Notion 只作公盤鏡像入口，不改寫 GitHub
- 每個入口項需保留 Source Ref
- 公盤若需更新，優先更新 GitHub 後再同步鏡像
- 首輪通過後，才可討論是否升為公盤主工具
