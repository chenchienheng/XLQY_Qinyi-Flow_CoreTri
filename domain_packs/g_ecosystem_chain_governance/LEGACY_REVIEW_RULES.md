## Red Doors

- Archive Candidate != Deleted.
- Moved Item != Reviewed Content.
- Folder Placement != State Approval.
- Old Document != Current Doctrine.
- Historical Value != Active Lane.

## Final Rule

Review first, reposition second. Do not let a folder location become silent approval.
