# Open XuanLing Core

Status: Candidate / Not Approved / No Runtime

Open XuanLing Core is a human-centered semantic governance protocol for AI-assisted work, tools, authority, and return paths.

It is not an AI agent framework, not a chatbot, and not a platform-specific automation tool.

It defines a governance chain:

```text
Source → Carrier → Authority → Gate → Action → Return → Rebuild
```

The first usable unit is Gate Decision: given a task or proposal, determine whether it should Go, Conditional Go, Park, Stop, or require Human Approval, then produce a Return Packet.

## What This Is

A small governance core for deciding whether an AI-assisted task, tool action, workflow proposal, or platform handoff may move forward.

## What This Is Not

- Not runtime.
- Not an agent framework.
- Not a chatbot.
- Not automation.
- Not a company policy.
- Not a platform adapter.
- Not a replacement for human approval.
