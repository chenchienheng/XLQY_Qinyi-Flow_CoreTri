# World Chain Master Axis

> Establishment of the World Chain as one of the five primary Coretri axis layers.

---

## 1. Definition

The World Chain governs the external environment, ecosystem families, platform
conditions, public interfaces, anti-capture mechanisms, and external node
onboarding.

## 2. Scope

The World Chain explicitly covers:

- ecosystem families
- platform and tool positioning
- public and world-facing interfaces
- anti-capture and replaceability guidelines
- external node onboarding pathways

This axis does **NOT** manage:

- low-level execution tasks
- isolated single-tool integration tasks
