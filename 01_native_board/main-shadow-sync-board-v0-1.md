# Main Shadow Sync Board v0.1

一句核心：
此板不是 main 的替代品，而是讓 seed 中已成形、但尚未併幹的主骨，先以穩定摘要的方式被其他窗口與主幹讀到。

## 0. 狀態
Draft v0.1

## 1. 使用方式
- 每一項只放穩定摘要
- 來源仍以 seed source path 為準
- main 讀者可先由此板判定是否需要下探 seed

## 2. 欄位
- Item ID
- Title
- One-line Summary
- Seed Source Path
- Current Status
- Merge Readiness
- Read Order
- Notes

## 4. 原則
- Shadow board 只放可穩定摘要的內容
- 不把 seed 全量細節搬進 main 視線
- 任何摘要都必須帶 seed source path

## 5. 收斂語句
main 若要接上 seed，不應直接吞掉 seed 全量變動；先由 shadow board 提供穩定摘要，其他窗口才有清晰入口，主骨才不會因讀取落差而模糊。
