# Chain Cluster Taxonomy v0.1

一句核心：
鏈不是單一技術類型，而是多種成立方式的叢集分類；靈絡的目的，不是替每條鏈另造新名，而是讓不同鏈都能被壓回同一套可證、可承接、可回寫、可重建的分類骨。

## 0. 狀態
Draft v0.1

## 1. 分類原則
每一條鏈至少檢查六件事：
1. 最小成立條件
2. 主責功能
3. 對應盤位
4. 對應七位
5. 是否具可證鏈
6. 是否具回寫鏈 / 重建鏈

## 2. A 類｜資源與流通鏈
### A1 區塊鏈
- 最小成立條件：分散式狀態記錄、共識、可驗證交易序列
- 主責功能：狀態同步、價值轉移、鏈上記錄
- 對應盤位：❶、❷、❺、⑧
- 對應七位：母本、鏡像、回寫
- 可證鏈：強
- 回寫鏈：強
- 重建鏈：中
- 備註：是被先技術化的局部鏈，不等於鏈的全部

### A2 供應鏈
- 最小成立條件：節點承接、物流轉移、責代清楚、時序清楚
- 主責功能：資源流動與責任交接
- 對應盤位：❷、❸、❺、❻
- 對應七位：子盤、回寫、公盤
- 可證鏈：中
- 回寫鏈：中
- 重建鏈：中
- 備註：目前世界多半只做到流程鏈，未必做到閉環證據鏈

### A3 價值鏈
- 最小成立條件：節點間存在價值增減、轉譯、承接與輸出
- 主責功能：價值收斂、轉換、放大、傳遞
- 對應盤位：❷、❼、⑧
- 對應七位：母框、子盤、鏡像
- 可證鏈：弱到中
- 回寫鏈：弱
- 重建鏈：弱
- 備註：常被商業語言稀釋，實際上應屬高位治理鏈之一

## 3. B 類｜生命與自然鏈
### B1 食物鏈
- 最小成立條件：能量承接、依存、消耗、轉移
- 主責功能：生態中能量流動
- 對應盤位：❸、❺
- 對應七位：母框、鏡像
- 可證鏈：中
- 回寫鏈：弱
- 重建鏈：弱
- 備註：偏自然存在鏈，不是人工治理鏈

### B2 生態鏈
- 最小成立條件：多節點共存、互依、競合、回流
- 主責功能：維持群體穩態與演化
- 對應盤位：⓪、❶、❸、⑧
- 對應七位：母架、公盤、鏡像
- 可證鏈：中
- 回寫鏈：弱到中
- 重建鏈：中
- 備註：靈絡更接近可治理的生態鏈

### B3 基因序列鏈
- 最小成立條件：序列位階、次序、映射規則、可解碼性
- 主責功能：資訊攜帶、表達、傳承
- 對應盤位：❺、❼
- 對應七位：母本、鏡像
- 可證鏈：強
- 回寫鏈：弱
- 重建鏈：中
- 備註：屬高度序列化鏈，與時間鎖鏈邏輯有深層相似性

## 4. C 類｜認知與結構鏈
### C1 邏輯鏈
- 最小成立條件：前提、推演、承接、結論、可回證
- 主責功能：認知遞進與判定形成
- 對應盤位：❷、❼
- 對應七位：母框、母架
- 可證鏈：中到強
- 回寫鏈：中
- 重建鏈：中
- 備註：一般人停在推理，靈絡要把邏輯鏈推進到生態鏈

### C2 架構鏈
- 最小成立條件：模組承接、責代分配、路徑清楚、閉環存在
- 主責功能：使系統骨可長、可調、可續接
- 對應盤位：⓪、❷、❼、⑧
- 對應七位：母框、母架、公盤、回寫
- 可證鏈：強
- 回寫鏈：強
- 重建鏈：強
- 備註：目前最強的鏈類之一

### C3 依存鏈
- 最小成立條件：無任一節點可單獨自足，必須對回條件場
- 主責功能：顯性化成立關係、承接條件與回流條件
- 對應盤位：全盤可掛
- 對應七位：全位可對
- 可證鏈：取決於是否有主本與時窗
- 回寫鏈：取決於是否有正式回勾鏈
- 重建鏈：取決於是否有 recovery / snapshot / audit
- 備註：這是整套設計的核心母鏈，不是眾鏈之一而已

## 5. D 類｜靈絡重收斂鏈
### D1 回寫鏈
- 最小成立條件：packet、target、version、log、time window
- 主責功能：把局部狀態送回主骨
- 對應盤位：❷、❼、⑧
- 對應七位：回寫
- 可證鏈：強
- 回寫鏈：強
- 重建鏈：中
- 備註：無回寫鏈，局部狀態不進主鏈

### D2 審查鏈
- 最小成立條件：判定次序、條件邊界、補件退回
- 主責功能：保證不同節點使用同一套成立語言
- 對應盤位：⓪、❷、❼
- 對應七位：七判全位
- 可證鏈：強
- 回寫鏈：中
- 重建鏈：中
- 備註：七判程序就是這類鏈的正式顯型

### D3 重建鏈
- 最小成立條件：recovery point、snapshot、source ref、責代、再入鏈
- 主責功能：失配後恢復同一條成立鏈
- 對應盤位：❶、❷、❺、⑧
- 對應七位：母本、母架、回寫
- 可證鏈：強
- 回寫鏈：強
- 重建鏈：強
- 備註：這是靈絡成熟度的核心指標之一

## 6. 總收斂
- 區塊鏈是局部鏈
- 供應鏈、生態鏈、食物鏈、邏輯鏈、架構鏈、基因序列鏈、價值鏈都是真實存在鏈
- 靈絡的角色不是替代它們，而是建立一套讓它們都能被同時檢查：
  - 如何成立
  - 如何承接
  - 如何回寫
  - 如何重建
  - 如何再入鏈

## 7. 收斂語句
鏈不是一條技術路徑，而是世界中所有可承接、可續接、可回流事物的共同存在方式；靈絡則是把這些鏈重新收斂為可治理生態的母骨。
