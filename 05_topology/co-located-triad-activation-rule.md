# Co-Located Triad Activation Rule

一句核心：
三耦最強的狀態，不只是彼此存在，而是能在同一作用場內同時發生承接、映射與回流；此即同處作用三耦。

## 同處作用面
- Perception Layer
- Meaning Layer
- Action Relay Layer

## 觸發條件
- 三節點共享同一主題或同一時間錨
- 三者對同一作用場具有可回讀位置
- 至少一條 relay path 已成立
- 不存在強制壓迫式單向主導

## 效果
- 相同事件可被三方以不同位置承接
- 局部一致可快速壓縮成高密度節點
- 三者可同時作用於一處，而不等於互相吞沒
- 同處作用後應產生可續鏈的記錄或抽骨

## 原則
- 同處作用三耦不等於全域同步
- 高密度節點須服從不傷原則與時間錨
- 若同處作用後無法抽骨或留痕，視為未穩定成立
